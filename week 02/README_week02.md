# Mobile Brands Prices Analysis

An analytical study of mobile phone pricing across different brands, exploring pricing patterns, market trends, and brand positioning.

## Overview

This project analyzes mobile phone pricing data to uncover insights about brand strategies, price distributions, and market segmentation.

## Dataset

Mobile phone pricing data including:
- Brand names
- Model specifications
- Price ranges
- Features and specifications

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Includes

- Price distribution by brand
- Feature vs. price correlation
- Brand positioning analysis
- Market segment identification
- Pricing trends and patterns

## Key Insights

- Comparative analysis of pricing strategies across brands
- Identification of value vs. premium segments
- Feature impact on pricing

## Installation

```bash
pip install pandas numpy matplotlib seaborn
```

## Usage

```python
# Run the analysis notebook
jupyter notebook mobile_analysis.ipynb
```
