# Apps Prices and Ratings Analysis

Comprehensive analysis of mobile application pricing strategies and user ratings to identify success patterns in the app marketplace.

## Overview

This project explores the relationship between app pricing, user ratings, and download performance across different app categories.

## Dataset Features

- App names and categories
- Price points (free/paid)
- User ratings (1-5 stars)
- Number of downloads
- App categories
- Review counts

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Components

1. **Pricing Strategy Analysis**
   - Free vs. paid app performance
   - Optimal pricing points
   - Category-wise pricing trends

2. **Rating Analysis**
   - Rating distribution patterns
   - Correlation between price and ratings
   - High-performing app characteristics

3. **Category Performance**
   - Top-performing categories
   - Category-specific insights
   - Market opportunities

## Key Findings

- Relationship between pricing and user satisfaction
- Optimal price points for different categories
- Factors influencing app ratings
- Free vs. paid model effectiveness
