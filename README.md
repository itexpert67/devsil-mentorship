## About

Weekly data science projects built during the DEVSIL mentorship program. Each week covers a full pipeline — data cleaning, EDA, modeling, and reporting.

Week 09 introduces the first live deployable tool — AutoEDA — which automates the entire EDA process for any CSV dataset and recommends ML models, neural networks, and data fixes inline.
